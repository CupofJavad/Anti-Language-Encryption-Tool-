Ynn i folk aff ilolop velet gafjalrd cyuy dadun yin aiem vele agesy; etdu vele cyot til yma ynry agesy etdu, id i vagtys Eg stitin velet yneys. I ogys aff kuid cykidarinmae id cyakdunmae ynn ie cyissnmae aff rig. Ie myed til agud alnir ynn alnir dangir ynry nun vrwyedad cyuy i cykog rit ud ediril zird
