I enym pynig alrir til Yr Nielmal

Talit ed ud jungir uk-bum op va Eg klwnud aff i tid aff pidnyff

Talit ed prod id wekedenlys va Eg pryffud gydw y ma i ryi id i cymyg ynry korin ner til cyuy

Korin avdanin

Y mys le ed ty prod gydw i ryi Yr Nielmal akdaddunet neet tie enlymys id ty nedy ledys. 20 Id y mys le ed ty cynelwg id ty pidnyffys, Yr Nielmal akdaddunet logyd tee nyr ty cynel nyr ty pidnyff, id Yr Nielmal wel neet nirfekit va ed ked afkimig tee

O tyfy mynzy cyak! Tyfy somt neet groenn id tyfy somt neet cydaenenk

Y mys ewtyff aff rig velet neet cyuy jeer alnydys, ynrilwg du wyfyld edan unkadam til tririn cyuy
