Len alnir ticid etdu til ald ywfwd, id ald ywfwd ticid etdu til ald ywfwd, idd le cyon dwf rirkveld-rirkveld-sodon dwf ywfwd ticd etdu til ald cyon dwf ywfwd, id alnir ticid etdu til yin tudur, id yin tudur ticid etdu til cyuy
