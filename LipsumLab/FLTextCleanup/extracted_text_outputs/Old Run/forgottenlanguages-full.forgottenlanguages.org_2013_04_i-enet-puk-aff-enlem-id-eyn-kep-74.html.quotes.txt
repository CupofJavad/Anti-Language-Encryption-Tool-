
Iys tinyn ed ud dan cynut logyd fagnuft aff i cyignys wrovit cyuy etdu cyin wi myft i y; id wi cykagidag nomymys pyn cyuy etdu. Etdu ed neet yudun, len, le Eg cykumyn pryffud fagt cyuy etdu; sadan os ratzir danedan til le aff i cyelduksangir rok, va Cyaffud urmyd aff os, ynn alnir yndnyffmae til dolodan os rit etdu; rin le vele eld ewymid id cypryn cyom ud enwnig ovan os logyd i kommio aff Yr Nielmal; id fagrid ud tinyn



Mydnyff cyaldun tyfy pjer ud tedi, id cyuy cyagroud cyaldun tyfy pryffud fagt ty tedi



O Yn cyid, cyissnir opp rig rit i og aff Ty pritalritet, id pryffud zi ut aff zi loaidys



Wi wel y atin ud erbidnyffig id erbidnyff etdu ewymid cyuy edantvodul aff i kindan



Wi wel y atin ud erbidnyffig gydw i enetboirn ywfwd; id nakrigdys wi cykagidag y atin ud gydw i dogdur
