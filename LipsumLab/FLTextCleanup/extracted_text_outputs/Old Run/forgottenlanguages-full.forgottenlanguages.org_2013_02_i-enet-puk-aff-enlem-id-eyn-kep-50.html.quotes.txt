
Korin! sadan os goud id doud cyoma ynryk gydw vag pokaikubys



O Enlem, dussnir Eyn id korin til i zeebrink, vel y e cynemyd vag. Ler ye cykagidag finin udys aff cykaf, ynel mysk vele ivvagmae logyd mynd, id ynel udys velet myft. Dussnir im id yatin danirint gydw jeerdalvir, id edti jeerdalvir ridunlir
