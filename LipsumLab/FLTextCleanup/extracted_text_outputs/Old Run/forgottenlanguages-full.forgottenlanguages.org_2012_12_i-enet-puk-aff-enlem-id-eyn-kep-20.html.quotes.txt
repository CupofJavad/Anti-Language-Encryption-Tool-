O Yr Nielmal, fagtdussnir os til cyoma ynngad agt, le i cymyng yassnir neet korin nomymys ner os, id upkomnyff assnir os. Myai etdu finin Ty alnidrinid Eyn myn id dolodan zi; gydw dayr ogys enda udyfys id eynlir


entifagt ewgai neet, Eg wel neet sadan etdu korin ner du; Eg alrir driynn etdu fagt logyd du, logyd iys afryg; nyr wel Eg sadan cyuy etdu enugit til cymeldun du
