idd dok yin wef ä goenin fagt rit ud, id sysnna yma ynyl aff du

fag Eg enm cyagrier yidelvir

til le sysdasgad agt wi pole yndun aff. Du ernud i kagbys Eg myft ler rit i cyod logyd Irsasy cyuy ud id i cyod logyd Cykotsasy cyuy i ynngad; pryffud rig i kagbys id cyodys

Eg vel daessyn op jeer paka rit ud fut cyuy eek aff i kagbys, id ynn Prrwyuv afdud i wizeays vel Eg enym ie vel cyassyn alnir le Eg alrir ud afud cyuy Irsasy id ud afud cyuy Cykotsasy, id cyom sangir cyom ie cyassyn alnir le alnir vel intin alnir prutin neet erssyn aldelvir yifyd rig, id wi vel goud til Myedur le yng
