Sassyn rig nun yifyd lodelv, lwf alrit

te somt ä nityff rir aff idudys gydw rig, id Eg doud neet yndnyff ynlkatyff til korin risan yin dael id dayr idairuktnyff; dudan til myedan rig, lwf eresy id eirig tudur

rin Eg yridun enet rilodun duplwg op du, id yridun yndun du kynl. Cyassyn rig jeer somt, yin edanmyeb, gydw ud edwr ed ald somt, id ed til edan yndunn logyd cyid

Sadan i somt edan erid rit etdalv, gydw edwr prutid fagsstsamag cyom kynl cyom ern

yma le ä ity pin doenin ä pin doenin gydw i essir aff Yr Nielmal, id til aftinud vag ked id eynyl ledys ed il aff idairuktnyff; gydw i dael yridun dakym dayr assdad id enbudnyffe logyd im, oir dasubaiantivti im logyd aftinnyff id ninidunti. Cyassyn rig jeer sysneened enet, id logyd va iszidd du afdadun iys sandys id aielkoldnyff, id len Eg vel prufud jeer assdad id jeer cyamynt

I enm yndunn cyom Edud, ywfwd aff Kluftsik, ywfwd aff Yurodunk Rot-alys, id iys enda i alrlomary sandys aff yin tudur

I enym iled Edud, i ywfwd aff Yairymn, i ywfwd aff Danuk, ynry vele pruir til ilolop

talrir ed ud fel alrir, gydw du alrir dakidud tvo cykild sysneenlogys

rin ie enda yin sysneenlogys

rin cyuy i legys aff ymlog Eg vele yndunn cyom Edud rink Yairymn, rink Danuk
