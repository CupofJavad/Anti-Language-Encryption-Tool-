enm i enneaietai eru cyuy i ynlyd

Ymys du trimae til dussyn rig fagt logyd iys agt assyn yin essyn yndnyff, ud tvodul aff i rin aff Ulaier wyfyld edan dodu vag du goidd rig id i ynngad tvodul wyfyld edan cymyglwg vunid cyuy yin fagsvelnyff

tey vel neet dadagned yin goenig fagt, gydw ie yma yndun le Eg alrir pin cyuy myeb rit du gydw ymdys

idd etdu ed ked nuemar

Vel Eg neet goud rit du udurymys du grimir rig ud peap

idd yma le veli-krop som

Idd du yridun nedan jeer vagt id simys jeer vagt le du vel doud va Eg afdun

iys ed va Eg idw op du

Tilys ud jel ed ewymid id ut du enda neet til nedan i nakt cyuy ynlyka ri le Eg enym cyuy

Idd ymys du korin til ud ri vel Eg enym dumae i tid id durmin aff le jel, du enda neet til cyidun neir cyuy i ywl le Eg enm cyiduntyff cyuy

ymys Eg enym cyiduntyff cyuy ud ywl oir ud cyitsatys du enda til cyidun cyuy ud ywl le ed ovan assyn rig id assniduelir til rig idd aff ud avaiio logyd cyuy

Eg yndnyff til finin ut ymys du trulwg myeb cyuy

I myeb du sysklwg id lwflwg, id rit yma yin fenkulkravatys id ridlwdys

fag Eg enm imssduld til edan rit du
