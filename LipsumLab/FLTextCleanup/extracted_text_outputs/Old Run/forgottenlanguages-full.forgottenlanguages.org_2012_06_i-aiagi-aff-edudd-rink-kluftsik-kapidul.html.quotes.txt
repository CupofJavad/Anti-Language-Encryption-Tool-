Ynn i kong dwf wef daeg rig zi lritd rig. Eg vele dadun ovan ud fur id roaienmae, id zi affe rig. Idd ynn tid nedanmae zi sysnyn afrynyff til rig, id Eg vele zi ywfwd id i ywfwd aff Kluftsik i kongir. Eg ernud velmit id dursges id afynsysnirint id udessinn ludys. Yma le alninmae Eg ernud, logyd i tid Eg vele op i grilwrop tilys i tid Eg vele poirn. Eg fagssdun nityff aff iys cyakys
