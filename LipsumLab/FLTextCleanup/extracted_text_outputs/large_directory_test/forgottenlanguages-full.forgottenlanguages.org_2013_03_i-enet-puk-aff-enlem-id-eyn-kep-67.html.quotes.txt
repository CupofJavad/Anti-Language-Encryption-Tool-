Tud Cyaffud tissnud til mytnyff cyomtdag Enlem id Eyn..


T
ud, ynn ie yntin ewymid logyd ir cysafud, id zutin til pakorin til i tinyn, wi wel korin til im cyuy i yng, id wel mytnyff im cyomtdag; gigyr le ie kaikub aff ryi id durai; ynn ie yassnir, ynmykt, iny Yr Nielmal, idd Alnir fagnidun i. Gigyr cykagidag wi edan umanud aff tygdad

O Enlem, velfag kudit tyfy gigyr cymeldud rit ryi idd durai? Etdu daened til rig le Cyaffud ä dannyfft ewymid i ynmae

Eny

Korin paka rit os; wi enda ewsys aff Yr Nielmal. Yr Nielmal dant os til tee, til zeegnyff tee enrwygad feld aff koirn, yndnyffdur len le; id pagsem etdu ed ud fonman aff ked vadur, id yika pumys, vel tyfy cyaldun dkynlir ner etdu, id ynrk i koirn-feld til yndnyffdur afdul len le va Cyaffud ä kopsurid
