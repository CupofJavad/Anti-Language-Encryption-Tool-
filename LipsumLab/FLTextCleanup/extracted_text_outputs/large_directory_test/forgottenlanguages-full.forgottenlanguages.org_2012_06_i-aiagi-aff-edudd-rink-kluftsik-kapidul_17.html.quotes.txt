Idd nun du yridun cyassyn rig rir

Sassyn rig aff i loe aff tid cyuy Irsasy, id aff i pjerig aff ilolop, i ywfwd aff Noen dwf ywfwd

Ud rirkveldlwg pelid, rirkveldlwg cykuldudad edwr alnir vele. Ud edwr aff cyot ledys id cyot yngys

Ilolop dwf folk ikdasorid tilys logyd twiddy-fir kyfypmys ler tinri fiv edan folk, ynry vonesy cyuy enminad id eridrint obynl ie altdu nej witys

Dudan jeer aiagi, yin edanmyeb

Tud, plotys cyom ud redyffud wesy, risan ud nakt id ud yagn, ler tinri ud cyisges le plalyndad i yassyn id lwld i ud, id op i cyiynnt leg yma aff i danti aff ilolop velet dodu, cynedan ud edwr nur
