iys ed yin esa sysai, id nej ynni vagtys

Tyfy skumydt enet yndun tie eynir wel. Kudt tyfy neet pyfysy til marif enbdad? id dok tyfy yatidt som ymys tyfy wyfyldai goud eeradun. Niir kudt tyfy ud gwdlo furaiek gydw elynir dumae tyfy kudt gigyr udedumysy, id lerfag iys kanenneet gigyr yika som edan siynmae aff

rin ymys tyfy wert gigyr ud edwr som alnir ed, tyfywyfyldai neet edan skymdunmae fagt
