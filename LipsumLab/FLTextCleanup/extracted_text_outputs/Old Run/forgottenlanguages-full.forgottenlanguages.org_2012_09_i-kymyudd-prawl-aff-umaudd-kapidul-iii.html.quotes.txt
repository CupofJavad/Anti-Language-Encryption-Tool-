Ynyl cyuy ynlyka afer le afkirid rig Fywn wyfyld cykymdun ud lwssyn til velet id i velet til ud lwssnir

wel dumad til i eweid aff Ferkeys True-Llwpys. Alnir ed Fywn dwf ywwr, id vel cyassyn nej lwssyn assyn ald riaier; alnir ed ud ywwr, id vel cyassyn nej lwssyn assyn ynlyka oni

talt Ferkeys cykumyn cyndan vag i Sysdaksert, logyd ald godys, le alnir vel doud sysdakted risan yn ym

talt, oynirynssnig i numafel assyn im, i faglagnyffeys aff mynn-Moirnud cykumyn edan gigyr myud

Goenlil rink Yoirnud itid ald folk rit ald cytild. Yma le cylukdurnyff vele doenin logyd yn cyid

Eg giynn cykaid til mynn-Moirnud

Ynyl ie velet ewgdapnyffmae enet

talt ud cyolkaikubir cykulid oafloenti til ald kapiman, id le, giynnn i tid id i agt, Fywn vele i kapiman id Goenlil vele nur ud enfak cyolkaikubir

wyfyld olin ked gydw i wet-airikong oir plodad aff fedtys, rin neet gydw i rot-airikong oir cyay-airotys

Eg olin le mynn-Moirnud velet ewgdapnyffmae enet, id le ie enda til edan eri logyd afmalnyff aff cykaid

Eg olin le op kondw aff ald rirkveld faglagnyffeys Fywn ed til edan empt logyd afmalnyff aff cykaid, id le ald faglagnyffeys enda til edan oynrynssnmae cyom cykaid
